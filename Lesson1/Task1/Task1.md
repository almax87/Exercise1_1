#Task1

Данный метод получает на входе указатель на массив и его размер.
Суммирует все элементы массива и возвращает результат.

Асимптотика:
O(n) - по времени (т.к. при увеличении массива на n единиц, количество шагов увеличится также на n единиц)
O(1) - по памяти (т.к. при увеличении массива кол-во памяти, выделяемой под переменные без учета входных данных, не изменится)
